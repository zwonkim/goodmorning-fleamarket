# UI Guideline

## Design Principles

-   Mobile-first
-   Consistency over uniqueness
-   Reusable components
-   Bold clean outlines
-   Flat colors
-   Rounded corners
-   Minimal shadows

## Shadow

0 2px 8px rgba(0,0,0,.06)

## Empty State

Every major page should provide an empty state.

Each empty state includes: - Official mascot SVG - Friendly title -
Helpful description - Primary CTA

## Loading State

Never display only a spinner.

Include: - Official mascot SVG - Short loading message - Subtle
animation (implemented in frontend)

## Error State

Every error page should include: - Official mascot SVG - Friendly
explanation - Retry action
