# Good Morning

## Overview

Good Morning is a private mobile-first flea market for a small group of
friends.

This is an MVP focused on simplicity.

## Tech Stack

-   Next.js (App Router)
-   TypeScript
-   Tailwind CSS
-   shadcn/ui
-   Supabase (Auth, PostgreSQL, Storage)
-   Vercel

## Core Features

-   Magic Link login (approved emails only)
-   Product upload
-   Product detail
-   Likes
-   Comments
-   My Page

## Out of Scope

-   Payment
-   Chat
-   Reservation
-   Search
-   Categories
