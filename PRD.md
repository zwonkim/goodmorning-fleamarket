# Product Requirements Document

## Product Vision

A warm and playful private flea market for friends.

## Authentication

-   Magic Link login
-   Approved email addresses only
-   First login requires nickname setup

## Product

Fields: - Images (3\~10) - Title - Price - Description - Condition

Condition values: - New - Lightly Used - Used

Users can: - Create - Edit own - Delete own

## Likes

-   Like / Unlike
-   One like per user

## Comments

-   Create
-   Delete own
-   No edit
-   No replies

## My Page

-   Edit profile
-   My Products
-   Liked Products

## Acceptance Criteria

-   Login succeeds with approved email
-   Products upload successfully
-   Likes update immediately
-   Comments appear chronologically
